# shopping

> Vue2.0+Vue-router+Vuex打造商城购物网站
> 浏览地址：http://1.15.157.83/story/vueShoppingcart/index.html
> 进入网页点击“手机"页面即可浏览模板

主要功能：

商品详情

商品添加购物车

商品购买

地址选择
